## Installation
Install GNU Stow
### macOS:
```bash
brew install stow
```
### Arch Linux:
```bash
sudo pacman -S stow
```
### Linux Mint (Ubuntu/Debian-based):
```bash
sudo apt update
sudo apt install stow
```
